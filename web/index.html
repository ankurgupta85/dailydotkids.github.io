Shared Classroom Device

One tablet per class — every teacher, zero password juggling

Many centers keep a classroom iPad at the check-in desk. Owners enable shared mode once — location admins with device permission can help tune session timeouts. Teachers switch with a quick PIN and the tablet locks when idle.

  • Owner-controlled setup: Individual devices or shared classroom mode, with configurable session and lock timeouts.
  • PIN teacher switching: Hand off the tablet with a 4-digit PIN — no re-entering email and password between shifts.
  • Clock in/out on shared tablets: Staff clock in before logging; owners review time logs from the admin dashboard.
  • Diary & record stay visible: Class notes and history stay browsable on the device — only logging and clock actions need the active teacher.

For Classroom Teachers

Teachers want to teach, not type. Log activities from a class roster, add diary notes, browse class history, generate end-of-day reports, and keep parents in the loop — on a personal phone or the classroom tablet.

  • 3-tap activity logging: Meals, naps, photos, moods, and notes — for one child or in bulk from the class roster.
